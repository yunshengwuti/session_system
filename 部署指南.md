**完全免费，自动启动，通过链接访问**

## 第一步：准备代码仓库

### 1. 安装Git（如果没有）
```bash
git --version
```

### 2. 初始化Git仓库
```bash
cd /Users/liyawen/Desktop/实习生日常/会话信息整理/session-system
git init
git add .
git commit -m "Initial commit"
```

### 3. 推送到GitHub
- 访问 https://github.com/new
- 创建新仓库（名称：session-system，Private）


## 第二步：部署后端到Render

### 1. 注册Render
- 访问 https://render.com
- 用GitHub账号登录（免费）

### 2. 创建MySQL数据库
- 点击 **New +** → **PostgreSQL**（MySQL免费版已停，用PostgreSQL替代）
- Name: `session-system-db`
- Database: `session_system`
- User: `session_user`
- Region: `Singapore` (离中国最近)
- 点击 **Create Database**
- 等待创建完成，复制 **Internal Database URL**

### 3. 部署后端服务
- 点击 **New +** → **Web Service**
- 连接你的GitHub仓库 `session-system`
- 配置：
  - **Name**: `session-system-backend`
  - **Region**: `Singapore`
  - **Root Directory**: `backend`
  - **Runtime**: `Python 3`
  - **Build Command**: `pip install -r requirements.txt`
  - **Start Command**: `uvicorn main:app --host 0.0.0.0 --port $PORT`

- 添加环境变量（Environment Variables）：
  ```
  DATABASE_URL = 你复制的Internal Database URL
  ZHIPU_API_KEY = 你的智谱API密钥
  ```

- 点击 **Create Web Service**
- 等待部署完成（5-10分钟）
- 复制后端URL：`https://session-system-backend.onrender.com`

### 4. 初始化数据库
部署完成后，后端会自动创建数据库表。

## 第三步：部署前端到Vercel

### 1. 修改前端API地址
编辑 `frontend/src/api/index.js`：
```javascript
const api = axios.create({
  baseURL: 'https://session-system-backend.onrender.com/api',  // 改成你的后端URL
  timeout: 30000,
})
```

提交代码：
```bash
git add .
git commit -m "Update API URL"
git push
```

### 2. 注册Vercel
- 访问 https://vercel.com
- 用GitHub账号登录（免费）

### 3. 导入项目
- 点击 **Add New** → **Project**
- 选择 `session-system` 仓库
- 配置：
  - **Framework Preset**: `Vite`
  - **Root Directory**: `frontend`
  - **Build Command**: `npm run build`
  - **Output Directory**: `dist`

- 点击 **Deploy**
- 等待部署完成（3-5分钟）
- 复制前端URL：`https://session-system-xxx.vercel.app`

## 注意事项

### 1. Render免费版限制
- 15分钟无活动会休眠
- 首次访问可能需要等30秒唤醒
- 每月750小时免费（够用）

### 2. 数据库迁移（PostgreSQL替代MySQL）

如果Render不支持免费MySQL，需要修改代码支持PostgreSQL：

**安装依赖**：
```bash
pip install psycopg2-binary
```

**修改 `backend/models/database.py`**：
```python
# 将MySQL连接改为PostgreSQL
DATABASE_URL = os.getenv("DATABASE_URL", "postgresql://...")
```

### 3. 自动唤醒（避免休眠）

**方案1**：使用UptimeRobot免费监控
- 访问 https://uptimerobot.com
- 添加监控：每5分钟ping一次后端URL
- 保持后端活跃

**方案2**：前端定时心跳
在前端添加定时请求，保持后端活跃。
